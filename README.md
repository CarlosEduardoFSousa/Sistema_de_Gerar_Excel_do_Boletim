# Sistema_de_Gerar_Excel_do_Boletim
Desenvolvi um sistema cujo qual tem como objetivo gerar um arquivo Excel onde o usuário controla o nome das matérias e as notas bimestrais ao gerar o arquivo Excel terá o nome das matérias suas médias bimestrais e final, status se o usuário foi aprovado ou não e quanto falta para ele passar na disciplina.
