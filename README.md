# Sistema de Geração de Boletins em Excel

Este projeto é uma ferramenta prática desenvolvida para facilitar a organização acadêmica, permitindo que o usuário tenha controle total sobre o lançamento de matérias e notas. O grande diferencial é a capacidade de exportar esses dados automaticamente para um arquivo Excel formatado, oferecendo uma visão profissional do desempenho escolar.

## 🚀 Funcionalidades

* **Personalização de Matérias:** Liberdade para cadastrar e gerenciar os nomes das disciplinas de acordo com a grade curricular.
* **Lançamento Bimestral:** Interface intuitiva para inserção de notas referentes aos quatro bimestres do ano letivo.
* **Cálculos Automatizados:** O sistema processa automaticamente as médias bimestrais e a média final de cada disciplina.
* **Análise de Status:** Identificação imediata se o aluno foi aprovado ou se ainda precisa de pontos para atingir a meta.
* **Exportação para Excel:** Geração de planilhas .xlsx detalhadas, contendo todas as fórmulas de cálculo e indicadores de desempenho.

## 🛠️ Tecnologias Utilizadas

* **Linguagens:** Python (Lógica de processamento), JavaScript, HTML e CSS (Interface e integração).
* **Biblioteca de Excel:** Integração com bibliotecas de manipulação de dados (Pandas/Openpyxl).
* **Conceitos:** Automação de planilhas, Manipulação de DOM e Lógica de Programação Acadêmica.
* **Ambiente de Desenvolvimento:** Visual Studio Code.
