# 📊 Sistema de Gerar Excel do Boletim

Este projeto é uma ferramenta desenvolvida para automatizar a exportação de dados acadêmicos para planilhas eletrônicas. O sistema extrai informações de boletins escolares e as organiza em arquivos `.xlsx`, facilitando o relatório e a análise de dados de alunos e notas.

## 🚀 Funcionalidades

* **Exportação Automatizada:** Transforma os dados do sistema em planilhas do Excel com apenas um clique.
* **Organização de Dados:** Estrutura nomes, notas, faltas e médias em colunas e linhas formatadas.
* **Geração de Relatórios:** Facilita a criação de documentos oficiais e históricos escolares.
* **Integração de Sistemas:** Demonstra a comunicação entre o software de gestão e ferramentas externas de produtividade.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** C#
* **Biblioteca de Integração:** Interop Excel / ClosedXML (ou biblioteca de manipulação de Excel)
* **Interface:** Windows Forms (WinForms)
* **Ambiente de Desenvolvimento:** Visual Studio
